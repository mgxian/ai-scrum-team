## 一、页面结构与模块内容

### 1. 导航栏
- **Logo 区域**：文字 `[LOGO]`
- **桌面端菜单**（≥768px）：水平展示首页、产品、定价、帮助,要跟据实际业务背景判断，例如企业内系统则不需要定价之类信息
- **移动端菜单**（<768px）：汉堡图标（☰），点击展开/收起菜单链接（纯CSS或简单JS实现）

### 2. 首屏（Hero）
- 主标题：`<h1>[主标题占位]</h1>`
- 副标题：`<p>[副标题说明文字占位]</p>`
- 两个行动按钮：
- 主要按钮：`[开始使用]`
- 次要按钮：`[了解详情]`
- 右侧装饰：Font Awesome 6 图标占位（如 `fa-cube`），尺寸大且醒目

### 3. 核心价值 / 功能亮点
- **网格布局**（Grid/Flex），显示 **4 个卡片**- 每个卡片包含：
- Font Awesome 图标（如 `fa-chart-line`、`fa-bolt` 等）
- 标题：`[功能标题一]` ~ `[功能标题四]`
- 描述：`[功能描述占位文字，约15字]`

### 4. 案例展示 / 用户场景
- **简单轮播**（至少2个 slide）
- 每 slide 内容：
- 客户名称：`[客户名称]`
- 效果数据：`[效果数据占位]`（如效率提升 40%）
- 按钮：`[查看案例]`
- 交互要求：
- 提供 **两个切换箭头** 和 **指示点（dots）**- 可手动切换，**无需自动播放**（JS 实现基础切换逻辑）
- 数据建议在 `<script>` 中用数组定义，便于替换

### 5. 定价方案 / 套餐对比（注意如果是企业内使用的系统，则不需要这个内容）
- 横向 **3 个套餐卡片**- 每个卡片包含：
- 套餐名称：`[基础版]`、`[专业版]`、`[企业版]`
- 价格：`[￥XX/月]`
- 核心特性列表：至少3项，使用占位文本（如特性一、特性二）
- **推荐套餐**（专业版）需通过额外类名实现高亮样式（边框/背景色差异）

### 6. 关于 / 团队介绍
- 展示 **3 位成员**- 每位成员：
- 头像：Font Awesome 用户图标（`fa-circle-user`）
- 姓名：`[姓名占位]`
- 角色：`[角色占位]`
- 简介：`[一句话简介]`

### 7. 底部（Footer）
- 行动引导：`[开始试用]`
- 联系信息：邮箱 `[contact@example.com]`、电话 `[+86 123-4567-8901]`、地址 `[某市某区某路]`
- 社交媒体图标：Font Awesome（fa-weixin、fa-weibo、fa-github、fa-linkedin 等）
- 版权声明：`© 2025 [公司名称]. All rights reserved.`

## 二、样式与交互强制要求

1. **布局**：完全使用 **Flexbox 或 CSS Grid**，严格遵循**移动优先**，响应式断点自定（建议 768px）。
2. **移动端导航**：
- 宽度小于 768px 时，水平菜单隐藏，显示汉堡图标。
- 点击汉堡图标展开/关闭垂直菜单（**纯 CSS 实现**：利用 `:checked` 或 `:target`；**或简单 JS**：监听点击切换 `display`）。
3. **文案标记**：所有可替换文案（标题、价格、姓名等）使用 **方括号 `[ ]`** 包裹，便于查找替换。例如 `[产品名称]`、`[功能描述一]`。
4. **代码注释**：
- 每个模块起始/结束处添加注释。
- 响应式断点处注明宽度。
- JS 逻辑关键步骤注释。
- CSS 自定义变量用途说明。
5. **无障碍基础**：
- 使用语义化标签（`<header>`, `<nav>`, `<main>`, `<section>`, `<article>` 等）。
- 图片图标设置 `alt=""` 或 `aria-hidden="true"`。
- 按钮添加 `aria-label`（如菜单按钮）。

---

## 三、数据与内容注入约定

- 案例轮播的数据**必须在 `<script>` 中以数组变量形式定义**（如 `const slides = [...]`），动态渲染至 HTML。
- 核心价值卡片、定价卡片、团队成员列表**可直接写静态占位 HTML**，但需在注释中注明此处数据可由数组循环生成。
- 所有 `[占位文本]` 均保持原样输出，**不要替换为其他文案**。

## 四、技术栈与依赖

- **仅原生三件套**：HTML5、CSS3、ES6。
- **允许外部库**：Font Awesome 6 免费版（通过 CDN 引入）。
- **禁止**：任何 CSS 框架（如 Bootstrap、Tailwind）或 JS 库（如 jQuery、React）。
