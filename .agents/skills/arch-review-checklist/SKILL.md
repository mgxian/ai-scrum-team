# Architecture Review Checklist

## Workflow

1. Read the complete architecture design (tech selection, domain model, API design)
2. Review against checklist dimensions:
   - Scalability and performance
   - Security and authentication
   - Data consistency and integrity
   - API design quality
   - Operational readiness (monitoring, logging, deployment)
   - Cost and maintainability
3. Flag issues by severity (high/medium/low)
4. Output review report

## Input

- Complete architecture design documents

## Output

- Architecture review report with issues and recommendations
