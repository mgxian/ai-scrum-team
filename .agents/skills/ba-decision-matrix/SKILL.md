# Decision Matrix

## Workflow

1. Read the requirement and identify complex decision points
2. Extract conditions and outcomes for each decision
3. Build decision matrix with all condition combinations
4. Validate completeness (no missing combinations)
5. Output as structured table

## Input

- Optimized requirement, scenario analysis

## Output

- Decision matrix tables
