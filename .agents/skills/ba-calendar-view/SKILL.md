# Calendar View & Resource-Time Model

## Workflow

1. Read the requirement for resource scheduling
2. Design calendar view layout (day/week/month)
3. Define resource allocation rules
4. Design view interactions (drag, resize, filter)
5. Output calendar view specification

## Input

- Requirement document, resource types

## Output

- Calendar view specification with interaction rules
