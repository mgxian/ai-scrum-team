# Validation & Business Rules

## Workflow

1. Read the requirement and data model
2. Identify all validation points (frontend and backend)
3. Define validation matrix per form/entity
4. Define backend-specific validation rules
5. Define numbering generation rules if applicable
6. Output structured rule tables

## Input

- Optimized requirement, data model, function list

## Output

- Validation matrix, backend check rules, numbering rules
