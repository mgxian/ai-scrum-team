# Scenario Analysis

## Workflow

1. Read the optimized requirement, roles, glossary, and process analysis
2. Identify all business scenarios (normal, CRUD, config)
3. For each scenario, identify key nodes and decision points
4. Apply scenario-guided detail analysis for complex scenarios
5. Output structured scenario list

## Input

- Optimized requirement document
- Role list, glossary, process analysis results

## Output

- Scenario list with key nodes and types
- Detailed scenario analysis for complex scenarios
