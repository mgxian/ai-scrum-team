# System Architecture Design

## Workflow

1. Read the requirement doc and tech selection results
2. Design module decomposition and responsibilities
3. Define communication patterns between modules
4. Design deployment topology
5. Generate system context and container diagrams

## Input

- Requirement doc, tech selection results

## Output

- System architecture document with diagrams
