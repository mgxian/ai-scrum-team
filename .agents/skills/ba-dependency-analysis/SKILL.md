# Dependency Analysis

## Workflow

1. Read the optimized requirement and scenario analysis results
2. Identify external system dependencies (APIs, data feeds)
3. Identify internal module dependencies
4. Assess dependency risks and mitigation strategies
5. Output dependency matrix

## Input

- Optimized requirement, scenario analysis

## Output

- Dependency matrix with risk assessment
