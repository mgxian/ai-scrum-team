# Business Glossary

## Workflow

1. Read the optimized requirement document
2. Extract all domain-specific terms and business concepts
3. Define each term with precise, unambiguous descriptions
4. Identify synonyms and related terms
5. Output as structured glossary table

## Input

- Optimized requirement document

## Output

- Glossary table (term, definition, synonyms, context)
