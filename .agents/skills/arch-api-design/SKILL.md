# API Interface Design

## Workflow

1. Read the function list, domain model, and role-permission matrix
2. Design API endpoints per module
3. Define request/response schemas
4. Specify authentication and authorization per endpoint
5. Define error response format and codes
6. Output API design document

## Input

- Function list, domain model, role-permission matrix

## Output

- API design document with endpoint specifications
