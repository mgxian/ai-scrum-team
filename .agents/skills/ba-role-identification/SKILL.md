# Role Identification

## Workflow

1. Read the optimized requirement document
2. Identify all user roles involved in the system
3. Define each role's responsibilities and permission scope
4. Map role interactions and hierarchies
5. Output structured role list

## Input

- Optimized requirement document

## Output

- Role list with responsibilities and permissions
