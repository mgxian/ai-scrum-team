### **一、表单结构&布局**1. **单列布局** -垂直排列所有表单项，标签在上、输入框在下。

-
适用于移动端或步骤简单的表单。

2. **多列布局** -并排排列相关字段（如姓名拆分为姓和名）。

-
注意响应式适配（在小屏自动转为单列）。

3. **分组与分割** -用卡片、分割线或标题对表单项分组（如个人信息敁支付信息）。

4. **标签对齐方式** -顶对齐标签（易扫描，适合国际化）、左对齐（节省垂直空间）。

### **二、表单元素类型**1. **基础输入框** -单行文本（如姓名、邮箱）、密码输入（隐藏字符）、多行文本（备注）。

2. **选择类控件** -单选按钮（互斥选项）、复选框（多选）、下拉菜单（节省空间）。

3. **开关/滑动选择** -用于二选一设置（如开启通知）。

4. **按钮** -主按钮（提交）、次按钮（重置/取消）、禁用状态（验证未通过时）。

### **三、交互与反馈**1. **占位符提示** -在输入框内显示示例格式（如请输入手机号）。

2. **实时验证** -输入时立即验证格式（如邮箱后缀检测），错误时显示红色边框和提示文字。

3. **必填项标记** -用星号（*）或必填标签标注。

4. **操作反馈** -提交成功/失败提示（如Toast消息或页面跳转）。

### **四、视觉与可访问性**1. **间距与层次** -表单项之间保持统一间距，标题和分组使用不同字号/颜色。

2. **焦点状态** -输入框聚焦时显示高亮边框或阴影。

3. **无障碍设计** -为屏幕阅读器添加标签关联（`aria-label`）、错误提示的语音反馈。

### **五、特殊表单类型**1. **分步表单** -显示进度条（如第1步/共3步），允许返回上一步。

2. **搜索表单** -结合搜索图标和清除按钮，支持实时搜索结果展示。

3. **筛选表单** -紧凑排列筛选条件（如价格范围滑块、多选标签）。
