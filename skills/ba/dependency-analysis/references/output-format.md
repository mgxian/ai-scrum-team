# 输出格式

## 依赖关联矩阵

| 本系统功能 | 业务场景 | 所需依赖能力 | 对应的依赖系统 |
|-----------|---------|------------|--------------|

## 关键接口规范

每个关键依赖单独一节：

### 依赖系统名称：<名称>

- **接口功能描述**：
- **接口通信方式**：
- **请求交互规范**：
  - 本系统需提供的参数：
  - 参数示例与说明：
- **响应与结果约定**：
  - 成功响应：
  - 失败响应：
- **外部系统的输入校验规则**：
  1. 基础必填校验：
  2. 数据格式与类型校验：
  3. 业务逻辑校验：

### 规则

- 关联矩阵覆盖所有功能与场景
- 接口规范只针对关键依赖，非关键依赖在矩阵中列出即可
